## Experiment Title

Experiment 5

## Aim

To write a program to demonstrate tuple and related functions in Python.

## Algorithm

1. Define a function `calc(a, b)` that returns the addition, subtraction, multiplication, and division of `a` and `b`.
2. Read two integers `a` and `b` from the user.
3. Call the function `calc(a, b)` and store the returned values in `add`, `sub`, `mul`, and `div`.
4. Print the addition, subtraction, multiplication, and division results.

## Output

```text
Enter a: 10
Enter b: 5
Addition: 15
Subtraction: 5
Multiplication: 50
Division: 2.0
```
