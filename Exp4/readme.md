## Experiment Title

Experiment 4

## Aim

To write a program to demonstrate list and related functions in Python.

## Algorithm

1. Initialize a list `scores = [102, 45, 78, 12, 56]`.
2. Display the original list.
3. Add a new element `89` to the list using `append()` and print the updated list.
4. Insert the value `34` at index `2` using `insert()` and print the list.
5. Add multiple elements `[67, 23]` to the list using `extend()` and display the list.
6. Create a copy of the list using `copy()` and print the copied list.
7. Display the elements of the list.
8. Remove all elements from the list using `clear()` and print the empty list.

## Output

```text
Original Scores: [102, 45, 78, 12, 56]
After append: [102, 45, 78, 12, 56, 89]
After insert: [102, 45, 34, 78, 12, 56, 89]
After extend: [102, 45, 34, 78, 12, 56, 89, 67, 23]
Copied Scores: [102, 45, 34, 78, 12, 56, 89, 67, 23]
Scores: [102, 45, 34, 78, 12, 56, 89, 67, 23]
After clear: []
```
