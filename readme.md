## Python Lab Experiments

This repository contains a set of Python lab experiments. Each experiment is organized in a separate folder that includes the program file and a corresponding README.md with detailed information.

### Structure

- **Experiment 1**: Demonstration of different data types and operators in Python.
- **Experiment 2**: Implementation of control flow and looping statements in Python.
- **Experiment 3**: Demonstration of various string operations in Python.
- **Experiment 4**: Demonstration of list operations and related functions.
- **Experiment 5**: Demonstration of tuple usage and related functions.
- **Experiment 6**: Demonstration of dictionary operations and related functions.

Each experiment folder includes:

- `experimentX.py` – the Python program file.
- `README.md` – documentation containing the **Experiment Title**, **Aim**, **Algorithm**, and **Output**.
